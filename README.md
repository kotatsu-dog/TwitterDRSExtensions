# TwitterDRSExtensions
## 機能
- 特定ユーザーのツイートを期間指定で検索できます。

## インストール方法

1. このリポジトリをクローンまたはZIPダウンロード
<img src="./step1" alt="ダウンロード方法" />
2. ChromeでURLバーに `chrome://extensions/` を入力し、開く
3. 画面右上から「デベロッパーモード」をONにする
<img src="./step3" alt="デベロッパーモード" />
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択
<img src="./step1" alt="パッケージ化" />
