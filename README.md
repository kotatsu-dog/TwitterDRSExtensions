# TwitterDRSExtensions
## 機能
- 特定ユーザーのツイートを期間指定で検索できます。

## インストール方法

1. このリポジトリをクローンまたはZIPダウンロード（緑色の「Code」ボタンを押し、Download ZIPというところを押すとできます。）
2. ダウンロードしたZIPファイルを解凍します。
3. 好きな場所に移動します。
4. ChromeでURLバーに `chrome://extensions/` を入力し、開く
5. 画面右上から「デベロッパーモード」をONにする
6. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

わからなければ<a href="https://jackmochi.com/extension-dev/">こちら</a>をご覧ください。
